# 国标麻将算番 微信小程序

胡牌之后拍照识别手牌，基于 1998 版国标麻将规则自动计算番数。

## 功能

- **📷 拍照识别** — 对准排列整齐的14张手牌拍照，上传至云函数进行 AI 识别
- **✋ 手动输入** — 点选各花色牌面，支持圈风/门风/自摸等上下文设置
- **🧮 自动算番** — 手牌拆解引擎 + 81 种番种检查 + 不重复计分原则
- **📊 结果展示** — 列出所有命中的番种、番数和说明，是否达到8番起和标准

## 快速开始

### 1. 注册微信小程序

前往 [微信公众平台](https://mp.weixin.qq.com/) 注册小程序，获取 AppID。

### 2. 配置项目

编辑 `project.config.json`，将 `appid` 改为你的小程序 AppID：
```json
"appid": "wx-your-appid-here"
```

### 3. 开通云开发

1. 在微信开发者工具中，点击「云开发」按钮开通
2. 创建环境，记下环境 ID
3. 编辑 `app.js`，将 `env` 改为你的云环境 ID：
```js
wx.cloud.init({ env: 'your-env-id' });
```

### 4. 部署云函数

在微信开发者工具中：
1. 右键 `cloudfunctions/recognizeTiles` → 上传并部署：云端安装依赖
2. 等待部署完成

### 5. 图像识别接入（可选）

云函数默认使用 mock 模式返回示例结果。要接入真实 AI 识别：

- **百度 AI EasyDL**：训练麻将牌分类模型，替换 `cloudfunctions/recognizeTiles/index.js` 中的 `BAIDU_API_KEY` 和模型 URL
- **腾讯云图像识别**：使用自定义识别能力
- **自建模型**：部署 TensorFlow/PyTorch 模型到云服务器，通过 HTTP 调用

详细集成代码见云函数中的 `callBaiduAI()` 函数。

## 技术说明

### 牌面编码

每张牌用 0–33 的数字表示：

| 编号 | 牌面 |
|------|------|
| 0-8  | 一万～九万 |
| 9-17 | 一条～九条 |
| 18-26 | 一饼～九饼 |
| 27-30 | 东南西北 |
| 31-33 | 中发白 |

### 算番算法

1. **拆解** — 递归回溯尝试所有 (将牌 + 4面子) 的组合
2. **检查** — 对每种拆解方案检查全部 81 种番种
3. **去重** — 基于预定义的互斥表，移除被高番种蕴含的低番种
4. **取优** — 在所有拆解方案中取番数最高的结果

### 番种覆盖

当前已实现约 50+ 种常用番种的检查逻辑，其余番种已定义但检查函数为 TODO。番种列表和互斥关系见 `utils/constants.js`。
