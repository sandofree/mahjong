# 国标麻将算番 微信小程序

胡牌之后拍照识别手牌，基于 1998 版国标麻将规则自动计算番数。

## 功能

- **📷 拍照识别** — 对准排列整齐的手牌拍照，上传至云函数，由阿里云百炼视觉模型自动识别牌面
- **✋ 手动输入** — 点选各花色牌面（支持 14–18 张，含杠牌），设置圈风/门风/自摸/门前清/花牌等上下文
- **🧮 自动算番** — 递归拆解引擎 + 81 种番种检查 + 不重复计分原则，支持七对、十三幺等特殊牌型
- **📊 结果展示** — 列出所有命中的番种、番数和说明，标注是否达到 8 番起和标准
- **🚫 诈胡检测** — 无法构成合法胡牌结构的手牌返回 0 番并提示原因

## 快速开始

### 1. 注册微信小程序

前往 [微信公众平台](https://mp.weixin.qq.com/) 注册小程序，获取 AppID。

### 2. 配置项目

编辑 `project.config.json`，将 `appid` 改为你的小程序 AppID：
```json
"appid": "wx-your-appid-here"
```

### 3. 开通云开发

1. 在微信开发者工具中，点击「云开发」按钮开通
2. 创建环境，记下环境 ID
3. 编辑 `app.js`，将 `env` 改为你的云环境 ID：
```js
wx.cloud.init({ env: 'your-env-id' });
```

### 4. 部署云函数

在微信开发者工具中：
1. 右键 `cloudfunctions/recognizeTiles` → 上传并部署：云端安装依赖
2. 等待部署完成

### 5. 配置图像识别 API（可选）

云函数使用**阿里云百炼**的 `qwen-vl-max` 视觉模型识别牌面。未配置 API Key 时自动降级为 mock 模式。

1. 前往 [阿里云百炼控制台](https://bailian.console.aliyun.com/) 开通模型服务并获取 API Key
2. 在微信云开发控制台 → 云函数 → `recognizeTiles` → 环境变量中添加：

   | 变量名 | 值 |
   |--------|-----|
   | `DASHSCOPE_API_KEY` | 你的百炼 API Key |

3. 重新部署云函数使环境变量生效

> 主模型 `qwen-vl-max` 超时会自动 fallback 到 `qwen3-vl-flash`，无需额外配置。

### 6. 运行测试

算番引擎有独立的 Node.js 测试，无需微信开发者工具：

```bash
node test_engine.js
```

覆盖 23 个测试用例：碰碰胡、平和、七对、十三幺、大三元、小三元、清一色、九莲宝灯、四暗刻、诈胡检测等。

## 技术说明

### 牌面编码

每张牌用 0–33 的数字表示：

| 编号 | 牌面 |
|------|------|
| 0–8 | 一万～九万 |
| 9–17 | 一条～九条 |
| 18–26 | 一饼～九饼 |
| 27–30 | 东南西北 |
| 31–33 | 中发白 |

### 算番算法

1. **合法性校验** — 手牌必须能构成标准型（1 将 + 4 面子）、七对、或十三幺之一，否则判诈胡
2. **拆解** — 递归回溯枚举所有 (将牌 + 4 面子) 的组合，支持 14–18 张牌（超出部分为杠补牌）
3. **检查** — 对每种拆解方案执行已注册的番种检查函数
4. **去重** — 基于预定义的互斥表，移除被高番种蕴含的低番种；同名异番种只保留最高番
5. **取优** — 在所有拆解方案中取番数最高的结果

### 上下文选项

手动输入页支持以下上下文设置（影响圈风刻、门风刻、自摸、不求人、四暗刻等番种判定）：

| 选项 | 说明 |
|------|------|
| 自摸 | 是否自己摸牌和牌 |
| 门前清 | 是否无吃碰明杠 |
| 圈风 | 当前圈风（东南西北） |
| 门风 | 本家门风（东南西北） |
| 花牌 | 春夏秋冬梅兰菊竹，每张 +1 番 |

### 番种覆盖

81 种番种已在 `utils/constants.js` 中定义，其中约 50+ 种已实现检查逻辑。主要已实现番种：

- **88 番**：大四喜、大三元、绿一色、九莲宝灯、四杠、连七对、十三幺
- **64 番**：小四喜、小三元、字一色、四暗刻、一色双龙会、一色四同顺
- **48 番**：一色四节高、一色四步高、三杠
- **32 番**：混幺九、三风刻、一色三步高、全带五
- **24 番**：全大/全中/全小、一色三同顺、一色三节高、全双刻、清一色、三色双龙会、七对
- **16 番**：三色三同顺、三色三节高、三同刻、三暗刻、三色三步高、花龙、全带幺、清龙
- **12–1 番**：碰碰胡、混一色、五门齐、平和、断幺、一般高、喜相逢、连六、老少副、幺九刻、缺一门、无字、自摸 等

部分依赖牌局过程信息的番种（全求人、杠上开花、抢杠和、和绝张、双明杠等）因拍照场景无法获取而暂未实现。完整番种列表和互斥规则见 `utils/constants.js`。
